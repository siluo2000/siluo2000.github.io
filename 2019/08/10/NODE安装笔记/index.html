卸载node和npm

在安装nvm前要先卸载已经安装的node和npm。

命令行卸载的方法没看懂。可以找到自带的卸载程序卸载node。

安装nvm

1.NVM下载
2.下载完成后，将内容解压到C:\dev\nvm （需为全英文路径）
3.新建settings.txt文件：

root: C:\dev\nvm
path: C:\dev\nodejs
arch: 64
proxy: none
node_mirror: http://npm.taobao.org/mirrors/node/
npm_mirror: https://npm.taobao.org/mirrors/npm/

4.配置环境变量

  • window + r 打开运行面板，然后输入 : sysdm.cpl，打开系统属性面板
  • 在系统属性面板中切换到高级面板，点击下方的环境变量按钮，打开环境变量面板
  • 新建系统变量如下：
NVM_HOME: C:\dev\nvm
NVM_SYMLINK: C:\dev\nodejs
  • 在系统变量PATH中新增 %NVM_HOME%;%NVM_SYMLINK%;

5.打开cmd，检查是否配置成功

set NVM_HOME
set NVM_SYMLINK
nvm -v

安装node

命令：

# 查看所有版本
nvm ls

# 查看可安装版本
nvm ls-remote

# 安装最新版本
nvm install latest

# 安装随意版本
nvm install 4.6.0

# 切换node版本
nvm use 4.6.0

# 安装稳定版本
nvm install stable

# 查看当前版本
nvm current

配置node全局模块路径

  • 方法一：
    设置到自己想要放的位置：

打开：nodejs安装目录/node_modules/npm/.npmrc这个文件，修改里面的路径

prefix = E:\nodejs\node_global_modules
  • 方法二：

进入cmd，输入如下代码：

# 查看npm配置
npm config ls  

# 配置prefix修改node的全局模块安装路径
npm config set prefix E:\nodejs\node_global_modules
